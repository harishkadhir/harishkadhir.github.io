Driver Drowsiness Detection

YOLOv11 vs Real-Time Detection

This real-time AI system monitors drivers for signs of drowsiness, distraction, and unsafe behavior. Built with YOLOv11, OpenCV, and Pygame, it detects dangerous behaviors such as mobile phone usage, talking, head movement (left/right), and drowsiness, triggering an audio alert when detected.

Features

  • Real-time Detection using webcam input
  • Detects Drowsiness, Mobile phone usage, Talking, Head movement, Child presence
  • Audio Alerts via Pygame when distraction detected
  • Custom YOLOv11 model trained on driver behavior data
  • Runs on any device with a webcam

Tech Stack

  • YOLOv11 (Ultralytics for object detection)
  • OpenCV (Video frame processing)
  • Pygame (Audio alert handling)
  • Python 3.8+

Dataset

The dataset includes multiple classes for driver monitoring:

  • drowsy
  • mobile
  • talking
  • child
  • LeftRight (head movement)
  • person (ensures driver presence)

You can extend the dataset by adding new distraction classes to improve model accuracy.

Installation

git clone https://github.com/harishkadhir/Driver-Drowsiness.git

Install dependencies:

pip install ultralytics opencv-python pygame

How it Works

The system captures real-time video from the webcam and processes each frame using YOLOv11. Detected driver behavior is classified into categories like drowsiness, mobile usage, or talking. If unsafe behavior is detected, an audio alert is triggered via Pygame. It continuously monitors head movements and other indicators to enhance road safety.
