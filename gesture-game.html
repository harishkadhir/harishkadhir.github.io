Gesture-Based Boxing Controller

Control Big Shot Boxing with Hand Gestures

Control a boxing game using real-time hand gestures through your webcam! This project uses OpenCV, MediaPipe, and PyAutoGUI to detect your gestures and simulate key presses in the browser to control the game Big Shot Boxing.

Back to Portfolio GitHub

Features

  • Detects Left Punch, Right Punch, Uppercut, and Block gestures
  • Launches the boxing game in your browser automatically
  • Uses MediaPipe for accurate hand tracking
  • Uses PyAutoGUI to simulate game key presses
  • Real-time visual feedback via OpenCV
  • Gesture cooldown & threshold to prevent accidental triggers

How It Works

MediaPipe identifies wrist positions from both hands. Movement direction and speed are tracked frame-to-frame. Based on hand position and velocity, gestures are recognized and mapped to game key presses:

  • Left Punch: Left hand quickly moves right → Key ←
  • Right Punch: Right hand quickly moves left → Key →
  • Uppercut: One hand raised above the face → Key X
  • Block: Both hands close to the face → Key Z

Gesture Mapping

Gesture Key Press Detection Logic
Left Punch ← Left hand moves rapidly to the right
Right Punch → Right hand moves rapidly to the left
Uppercut X One hand raised above the face
Block Z Both hands close to the face area
